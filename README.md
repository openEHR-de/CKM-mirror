# German openEHR CKM Mirror

This is a mirror of the German HiGHmed Clinical Knowledge Manager (CKM) clinical models repository at https://ckm.highmed.org. 

It contains all active archetypes and templates from all CKM projects.
It includes both published assets and unpublished/unstable assets under active development on the CKM TRUNK.

This mirror auto-updates whenever the CKM TRUNK changes.
